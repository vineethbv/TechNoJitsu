Input Keypad with 8x8 matrix display project

  • EMO Project

This is a simple project meant to demonstrate the usage of 8x8 matrix display and a 4x3 input keypad with an Arduino. The project simply changes the image/pattern displayed according to the key pressed.

For this project, you will need:

  • MAX7219 matrix display
  • Jumper cables
  • Arduino UNO R3
  • 4x3 input keypad
Please note that we have only recomended the above components based on reviews and pricing and we do not expect you to stick to these components' models strictly.

To demonstrate this project our team has experimented using a 4x3 input keypad an MAX7219 8x8 matrix display. You can find other matrix displays as well, though this program focuses on an MAX7219 8x8 matrix display in particular. But the input keypad is always a standard input keypad. The alternative for this can be a 4x4 input keypad.

To start off with, the 4x3 input keypad consists of ROW1, ROW2, ROW3, ROW4, COLUMN1, COLUMN2, and COLUMN3, while an MAX7219 8x8 matrix display consists of Vcc, GND, DIN, CS, and CLK pins. So our connection diagram for this project with an Arduino (UNO in particular) goes as follows:

The connections made between ports 11, 10 and 9 to DIN, CS, CLK respectively can also be altered based on the requirements of your projects and isn't a fixed connection. As always, GND on any component is connected to GND on your Arduino board. Vcc is always connected to either 5V or 3.3V based on which component you use, and in this case it is connected to 5V. The connections for the input keypad however, can be altered based on one's convenience and can be connected to any of the digital pins excluding what is already used by the matrix display.

In order to program this board after making the necessary connections, you will need the Arduino IDE. If this is your first time using the IDE, then you will need to set up your board on the IDE. Please follow this link to set up your board if it is your first time using an Arduino: How to set up Arduino board. Once you have set up your Arduino board, you must include the necessary libraries for that particular project. In this case, you need LedControl and Keypad libraries for this project. Download the file and extract it to the same directory as your Arduino directory (By default C:\Users\usr\Documents\Arduino\libraries).

Now that we have everything set up, we can start programming our Arduino. You can find the code in our developer's GitHub repository: 4x3 Input Keypad Once entered, feel free to make any changes you wish while giving credits to the author for his contributions. After you made any / no changes to the code, save the file and keep in mind the file extension must be .imo as it is meant to be imported to your Arduino board. After compiling and uploading the file to your Arduino, you can notice a number pattern appears for a specified duration (in milliseconds) on your matrix display based on which key is pressed. And successfully completes this project.
