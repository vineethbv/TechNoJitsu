VRIABLE & CONSTANTS

A variable is nothing but a name given to a storage area that our programs can manipulate. Each variable in C has a specific type, which determines the size and layout of the variable's memory; the range of values that can be stored within that memory; and the set of operations that can be applied to the variable.
Variable names can contain alphabets numbers and underscore. Variable name never starts with a number and its case sensitive also. These are the predefined types :

TYPEDESCRIPTION
Char Typically a single octet(one byte). This is an integer type.
Int The most natural size of integer for the machine.
Float A single-precision floating point value.
Double A double-precision floating point value.
Void Represents the absence of type.

DECLARATION

A variable declaration provides assurance to the compiler that there is one variable existing with the given type and name. A variable declaration has its meaning at the time of compilation only, compiler needs actual variable declaration at the time of linking of the program.
The syntax for declaration is:

L AND R VALUE

lvalue: An expression that is an lvalue may appear as either the left-hand or right-hand side of an assignment.
rvalue: An expression that is an rvalue may appear on the right- but not left-hand side of an assignment.
In an assignment the lvalue comes at the left side and rvalue at the right side.
Ex: int g=20 ;
Here r value is 20 and the l value is the memory location/address value of the variable.

CONSTANTS

The constants refer to fixed values that the program may not alter during its execution. These fixed values are also called literals.
Constants can be defined by two methods:

  • Variable constant with help of const keyword
  • Constants with #define preprocessor

const

Variable basic data types like an integer constant, a floating constant, a character constant, or a string literal can be declared with the help of beow mentioned syntax.
Syntax : const data_type variable_name;

#include 
int main()
{
const int LENGTH = 10;
const int WIDTH = 5;
const char NEWLINE = '\n';
int area;
           
area = LENGTH * WIDTH;
printf("value of area : %d", area);
printf("%c", NEWLINE);
return 0;
}
Output: value of area : 50

#define

This is used in code for making general constant value where the value assigned to the quantity which could be anything.

#define pi 3.14159
