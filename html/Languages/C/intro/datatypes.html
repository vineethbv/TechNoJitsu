DATA TYPES

Data types refer to an extensive system used for variables or functions of different types. The type of a variable determines how much space it occupies in storage and how the bit pattern stored is interpreted.
Types of Data Types are :
Basic Types:
They are arithmetic types and consists of the two types: (a) integer types and (b) floatingpoint types.
Enumerated types:
They are again arithmetic types and they are used to define variables that can only be assigned certain discrete integer values throughout the program.
Derived types:
They include (a) Pointer types, (b) Array types, (c) Structure types, (d) Union types and (e) Function types.
We will see basic types in the following section.
